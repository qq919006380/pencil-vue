使用 HTML 片段

使用HTML片段

<template>
  <Button @click="openHTML">使用HTML片段</Button>
</template>

<script>
export default {
  methods: {
    openHTML() {
      this.$toast("<strong>这是 <i>HTML</i> 片段</strong>", {
        enableHtml: true,
      });
    },
  },
};
</script>

autoClose设置消息关闭时间，秒，设置为false则不关闭Boolean | Number—3
onClose关闭事件Function——
showClose是否显示关闭按钮boolean—false
enableHtml是否将 message 属性作为 HTML 片段处理Boolean—false
